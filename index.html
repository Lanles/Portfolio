Recent Projects

Project 1

Goblins And Dragons

A college project I made to resemble old-school RPG games using C# with some very basic mechanics. The player has the ability to make his own character or use the default one, and while roaming the dungeon he will randomly encounter different events from monsters to merchants and chests containing upgrades. In the end, he will encounter the dungeon boss.

Project 2

D&D Mapper

Old college project (a website made for displaying custom maps). It uses some basic PHP code for making accounts and uploading images to a server and also uses some basic JavaScript for displaying an image over a grid to use as a map for simple D&D gameplay.

Project 3

Running Game

This is a project from my online courses that uses C++ and the raylib libraries to create a very simple game. The game is supposed to imitate an endless runner, but it does have an end that is randomly generated, or better said the number of enemies drawn on screen is random. Project 4

Mini Top Down Game

Some of those elements are object collision, hero hp, very basic enemy AI and a few other little things.
